1. IIS 설치하기
 - 제어판 프로그램/기능, window 기능 켜기/끄기에서 설치할 애들 선택하기
 - 설치해야하는 애들은 블로그를 참고했음(https://blog.naver.com/leeks219/221856429796)

2. 홈디렉터리인 C:\Inetpub\wwwroot\ 디렉터리에 프로그램 파일들 넣어놓기

3. cmd 켜서 채팅 서버 실행시키기( java ChatServer )

4. 브라우저에 url 입력 ( http://localhost/(html파일) )
